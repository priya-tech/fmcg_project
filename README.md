**FMCG Project**

This is an end to end data engineering project using databricks. In FMCG domain, building an ETL pipeline for a real life industry use case where a big retail company acquires a small one and build the data pipeline to consolidate data from both the companies in a single lakehouse architecture. 
The tech stack of the data engineering project is: Python, Pyspark, Spark SQL, Medallion architecture, Databricks.
